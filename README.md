## Versioning

Versioning operates in a MAJOR.MINOR.PATCH format

After updating `VERSION` file to get template for changelog and update to gradle build:
- For MacOS run: `make update-version` in terminal
- For Windows run: `powershell -ExecutionPolicy Bypass -File update-version.ps1` in powershell
